ADBC driver is now the default driver for new Power BI connections
> ADBC ドライバーがこれからは新しい Power BI の接続のデフォルトのドライバーとなります。
February 20, 2026

New connections created in Power BI Desktop or Power BI Service now automatically use the Arrow Database Connectivity (ADBC) driver by default. Existing connections continue to use ODBC unless you manually update them to ADBC. You can still switch to ODBC drivers for new connections. See Configure ADBC or ODBC driver for Power BI.

>

Power BI デスクトップまたは Power BI サービスで作成された新しい接続は、デフォルトで Arrow Database Connectivity (ADBC) ドライバーを自動的に使用するようになりました。既存の接続は ODBC を継続して使用しますが、手動でADBC に更新することができます。新しい接続では ODBC ドライバーに切り替えることもできます。Power BI の ADBC またはODBC ドライバーの構成を参照してください。

Google Gemini 3.1 Pro Preview now available as a Databricks-hosted model
> Google Gemini 3.1 Proプレビューが現在Databricksでホストされているモデルとして利用可能になりました。
February 19, 2026

Mosaic AI Model Serving now supports Google Gemini 3.1 Pro Preview as a Databricks-hosted model.

To access this model, use:

  • Foundation Model APIs pay-per-token
    • Query reasoning models
    • Query vision models
  • Batch inference workloads using AI Functions
> モザイクAIモデルサービングは、DatabricksホストのモデルとしてGoogle Gemini 3.1 Proプレビューをサポートしています。 このモデルにアクセスするには、以下を使用してください: - Foundation Model APIs pay-per-token - Query reasoning models - Query vision models - Batch inference workloadsをAI Functionsを使って実行する
Dashboard visualization and interaction enhancements
> ダッシュボードの可視化と相互作用の強化
February 19, 2026
  • Pivot table hierarchies: You can now expand and collapse hierarchies in pivot tables.
  • Stacked bar chart labels: Stacked bar charts now support showing min/max label values and total labels.
  • Area and line chart sorting: Area and line charts now support sorting axes by measures.
  • Dashboard content search: Global search in Lakehouse and Databricks One now matches on dashboard content, not just titles.
  • Chart faceting: Charts now support faceting, rendering one chart per dimension value. See Faceting.
>
  • ピボットテーブルの階層: ピボットテーブルでは、階層を展開および折りたたむことができるようになりました。
  • 積み上げ棒グラフのラベル: 積み上げ棒グラフでは、最小値/最大値のラベル値と合計ラベルの表示がサポートされるようになりました。
  • エリアグラフおよび折れ線グラフのソート: エリアグラフおよび折れ線グラフでは、軸を指標でソートできるようになりました。
  • ダッシュボードのコンテンツ検索: Lakehouse およびDatabricks Oneでは、グローバル検索がタイトルだけでなくダッシュボードのコンテンツにも一致するようになりました。
  • チャートのファセット化: チャートではファセット化がサポートされるようになり、ディメンション値ごとに1つのチャートを描画できるようになりました。ファセット化をご覧ください。
Data Classification will soon be available by default for some workspaces with the compliance security profile enabled
> データ分類は、コンプライアンスセキュリティプロファイルが有効になっている一部のワークスペースでデフォルトで間もなく利用可能になる。
February 18, 2026

In mid-March 2026, Data Classification will be available by default for workspaces with the compliance security profile enabled and HIPAA controls selected.

> 2026年3月中旬に、コンプライアンスセキュリティプロファイルが有効化され、HIPAA管理が選択されたワークスペースでは、デフォルトでデータ分類が利用可能になります。

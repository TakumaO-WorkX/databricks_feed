Databricks JDBC Driver 2.7.5
> データブリックスJDBCドライバー2.7.5
October 23, 2025

Databricks JDBC Driver (Simba) version 2.7.5 is now available with the following improvements:

New features

The connector now supports Kerberos with proxy connections. To enable Kerberos proxy, set UseProxy=1 and ProxyAuth=2. To set proxy details, use ProxyHost, ProxyPort, ProxyKrbRealm, ProxyKrbFQDN, and ProxyKrbService.

Resolved issues

  • Fixed an issue where the connector failed to run complex queries that contained ? characters in native mode.
  • Fixed intermittent failures in Unity Catalog volume ingestion caused by unexpected connector behavior.
  • Fixed an assertion error in getColumns when a table included a column of type Void or Variant and the java -ea flag was enabled.
>

バージョン2.7.5のDatabricks JDBC ドライバー(Simba)が次の改善点を含んで利用可能になりました:

新機能

このコネクタはKerberosプロキシ接続をサポートするようになりました。Kerberosプロキシを有効にするには、UseProxy=1とProxyAuth=2を設定してください。プロキシの詳細は、ProxyHost、ProxyPort、ProxyKrbRealm、ProxyKrbFQDN、ProxyKrbServiceを使用して設定してください。

解決された問題

  • ネイティブモードで?文字を含む複雑なクエリの実行に失敗する問題を修正しました。
  • Unity Catalogボリューム取り込みの予期せぬコネクタの動作によって発生していた断続的な障害を修正しました。
  • java -eaフラグが有効になっている際に、テーブルにVoid型やVariant型の列がある場合のgetColumnsでのアサーションエラーを修正しました。
Zerobus Ingest connector in Lakeflow Connect (Public Preview)
> ゼロバスインジェストコネクタLakeflowConnect（パブリックプレビュー）
October 23, 2025

The Zerobus Ingest connector in Lakeflow Connect is in Public Preview. This connector enables record-by-record data ingestion directly into Delta tables using a gRPC API. See Zerobus Ingest connector overview.

>

Lakeflow ConnectのZerobus Ingestコネクタは、パブリックプレビューとなっています。このコネクタを使うことで、gRPC APIを利用して、レコードごとにデータをDeltaテーブルに取り込むことができます。詳細はZerobus Ingestコネクタの概要をご覧ください。

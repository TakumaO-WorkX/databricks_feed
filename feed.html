Databricks Docs Updates

Databricks Runtime maintenance update
> データブリックスランタイムメンテナンスアップデート
September 08, 2025

New maintenance updates are available for supported Databricks Runtime versions. These updates include bug fixes, security patches, and performance improvements. For details, see Databricks Runtime maintenance updates.

>

サポートされているDatabricks Runtimeバージョンに対する新しいメンテナンスアップデートが利用可能です。これらのアップデートには、バグ修正、セキュリティパッチ、およびパフォーマンス改善が含まれています。詳細については、Databricks Runtimeメンテナンスアップデートをご覧ください。

Databricks Apps support for Genie resources
> データブリックス・アプリのGenieリソースに対するサポート
September 08, 2025

Databricks Apps now supports adding an AI/BI Genie space as an app resource to enable natural language querying over curated datasets. See Add a Genie space resource to a Databricks app.

>

Databricksアプリは、選別されたデータセットに対する自然言語クエリを可能にするAI/BIジーニースペースをアプリリソースとして追加することをサポートしています。詳細はDatabricksアプリにジーニースペースリソースを追加するをご覧ください。

SAP Business Data Cloud (BDC) Connector for Databricks will soon be generally available
> SAP Business Data Cloud (BDC) Connector for Databricks は近日一般提供される予定です。
September 08, 2025

The SAP Business Data Cloud (BDC) Connector for Databricks is a new feature that allows you to share data from SAP BDC to Databricks and from Databricks to SAP BDC using Delta Sharing. This feature will be generally available soon.

>

SAP Business Data Cloud (BDC)用のDatabricksコネクタは、Delta Sharingを使ってSAP BDCからDatabricksへ、およびDatabricksからSAP BDCへデータを共有できる新しい機能です。この機能は近日中に一般公開される予定です。

Databricks connector in Microsoft Power Platform is in Public Preview
> Microsoft Power Platformのデータブリックス コネクタがパブリックプレビュー中です。
September 05, 2025

Use Databricks data to build canvas apps in Power Apps, flows in Power Automate, and agents in Copilot Studio by creating a Databricks connection in Power Platform.

See Connect to Databricks from Microsoft Power Platform.

> Databricksのデータを使って、Power Appsでキャンバスアプリを、Power Automateでフローを、Copilot Studioでエージェントを作成するには、Power Platformでのデータブリックスの接続を作成します。 「Microsoftのパワー プラットフォームからのデータブリックス接続」をご覧ください。
Serverless GPU compute now supports H100 accelerators (Beta)
> サーバーレスGPUコンピューティングがH100アクセラレータをサポートするようになりました(ベータ版)
September 05, 2025

Serverless GPU compute now supports on-demand H100 accelerators for single-node workflows and jobs. This feature is in Beta.

See Serverless GPU compute.

>

サーバーレスGPUコンピューティングはこのタイミングでH100アクセラレーターをオンデマンドで単一ノードワークフローおよびジョブに対応するようになりました。この機能はベータ版です。

詳しくはサーバーレスGPUコンピューティングを参照してください。

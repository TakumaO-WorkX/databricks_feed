Databricks Docs Updates

Trigger on update for pipelines (GA)
> パイプラインの更新トリガー (GA)
January 30, 2026

TRIGGER ON UPDATE is now generally available. Use trigger on update in Databricks SQL when creating a pipeline schedule to refresh the pipeline when a source table changes.

See Schedule materialized view refreshes and Change the schedule for a streaming table for more information.

>

UPDATE時のTRIGGERが一般的に利用可能になりました。Databricks SQLでパイプラインのスケジュールを作成する際に、ソーステーブルが変更された時にパイプラインをリフレッシュするためにUPDATE時のTRIGGERを使用してください。

マテリアライズドビューのリフレッシュをスケジュールするとストリーミングテーブルのスケジュールを変更するで詳細情報をご確認ください。

Lakeflow jobs system tables are now generally available (GA)
> レイクフロー ジョブ システム テーブルが一般利用可能になりました。
January 30, 2026

The lakeflow system tables jobs, job_tasks, job_run_timeline, and job_task_run_timeline are now generally available (GA). These tables provide insights into all jobs and job runs in your account.

For a reference of these tables, see Jobs system table reference.

>

湖流システムテーブル jobs、job_tasks、job_run_timeline、および job_task_run_timeline が一般提供 (GA) されるようになりました。これらのテーブルは、アカウントのすべてのジョブとジョブ実行の洞察を提供します。

これらのテーブルのリファレンスについては、ジョブシステムテーブルリファレンスを参照してください。

View and restore your recent workspace tab sessions
> 閲覧と復元最近のワークスペースのタブセッション
January 30, 2026

You can now view your past workspace tab sessions and restore them. See Switch between authoring contexts.

> 過去のワークスペース タブセッションを表示し、復元することができるようになりました。 オーサリングコンテキストの切り替えをご確認ください。
Paste images into notebooks
> 画像をノートブックに貼り付けます。
January 30, 2026

You can now copy images from your local file system and paste them into markdown cells in Databricks notebooks using ⌘ + V (mac) or Ctrl + V (Windows). See Display images.

>

ローカルファイルシステムから画像をコピーして、Databricksノートブックのmarkdownセルに⌘ + V(Mac)またはCtrl + V(Windows)でペーストできるようになりました。画像の表示を参照してください。

Dashboard visualization and data improvements
> ダッシュボードのビジュアル化とデータの改善
January 29, 2026
  • Fixed access request availability: Resolved an issue where the option to request access was not available and a "Resource not found" error was incorrectly shown.
  • Full-width pivot tables: Widgets now render pivot tables at full width for better readability.
  • Tooltip encodings on charts: Combo and dual-axis charts now support tooltip encodings.
  • Image rendering in pivot tables: Cells can now display images directly in pivot tables.
  • Dark mode scrollbar visibility: Scrollbars are now easier to see when using dark mode.
  • Increased pivot data download limits: Download limits now support up to 10,000 rows by 1,000 columns for pivoted data.
  • Fixed filter paste behavior: Resolved an issue where pasting a comma-separated list into a single-select filter only parsed the first element instead of keeping all values.
  • Fixed stacked chart label rendering: Labels on stacked charts with a quantitative x-axis now render correctly in notebooks and in the SQL Editor.
>
  • 固定アクセス要求の利用可能性: アクセスを要求するオプションが利用できず、「リソースが見つかりませんでした」というエラーが誤って表示されていた問題を解決しました。
  • 全幅ピボットテーブル: ウィジェットでピボットテーブルが読みやすいように全幅で表示されるようになりました。
  • チャートのツールチップエンコーディング: コンボチャートと双対軸チャートでツールチップエンコーディングがサポートされるようになりました。
  • ピボットテーブルでの画像表示: ピボットテーブルの中で画像を直接表示できるようになりました。
  • ダークモードのスクロールバー表示: ダークモードを使用する際にスクロールバーが見やすくなりました。
  • ピボットデータダウンロード制限の引き上げ: ダウンロードの制限が10,000行×1,000列まで対応するようになりました。
  • フィルターの貼り付け動作の修正: 単一選択フィルターにコンマ区切りのリストを貼り付けた場合、最初の要素のみ解析されていた問題を修正しました。
  • 積み上げチャートのラベル表示修正: 数量的なx軸を持つ積み上げチャートのラベルが、ノートブックやSQLエディターで正しく表示されるようになりました。

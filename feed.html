Databricks Docs Updates

Public URLs for ODBC driver downloads will be disabled on March 27
> 2023年3月27日にODBCドライバーダウンロード用の一般公開URLが無効になります。
March 03, 2026

On March 27, 2026, public S3 links for downloading the Simba Spark ODBC Driver will be disabled. After this date, scripts or processes that use direct public links to download the driver will no longer function. The driver will be available exclusively through the ODBC archive page, which requires sign-in.

To continue using automated downloads, migrate to the Databricks ODBC driver before March 27, 2026. See Migrate from Simba Spark ODBC Driver to Databricks ODBC Driver.

> 2026年3月27日に、Simba Spark ODBCドライバのダウンロード用の公開S3リンクが無効になります。この日以降、ドライバのダウンロードに直接公開リンクを使うスクリプトやプロセスは機能しなくなります。ドライバは、ログイン要件のあるODBCアーカイブページでのみ利用可能になります。 自動ダウンロードを続けるには、2026年3月27日までにDatabricksのODBCドライバへ移行してください。Simba Spark ODBCドライバからDatabricksのODBCドライバへの移行をご覧ください。

Agent Bricks: Knowledge Assistant is now available by default for Enhanced Security and Compliance customers
> エージェントブリックス:ナレッジアシスタントは、エンハンスドセキュリティおよびコンプライアンスのお客様向けにデフォルトで利用可能になりました。

Managed OAuth flows for external MCP servers
> 管理対象のOAuthフローのためのMCPサーバー
March 02, 2026

Databricks now provides managed OAuth flows for select external MCP servers, eliminating the need to register your own OAuth app or manage credentials. Supported providers include Glean MCP, GitHub MCP, Google Drive API, and SharePoint API.

See Install an external MCP server.

> Databricksは現在、特定の外部MCPサーバーに対して管理されたOAuthフローを提供しており、自身のOAuthアプリの登録や資格情報の管理を不要にしています。サポートされるプロバイダには、Glean MCP、GitHub MCP、Google Drive API、SharePointAPIが含まれています。 外部MCPサーバーのインストールについては、外部MCPサーバーのインストールを参照してください。
Notebook tagging (Public Preview)
> ノートブックタグ付け(パブリックプレビュー)
March 02, 2026

You can now apply tags to notebooks to organize and categorize them for easier management. Notebooks also support governed tags, including certification and deprecation system tags, to indicate trust levels or lifecycle status directly within the notebook interface. See Apply tags to notebooks.

> ノートブックにタグを適用して、管理を容易にするために整理およびカテゴリー化することができるようになりました。ノートブックには、証明書やライフサイクルステータスなどの管理されたタグもサポートされており、ノートブックインターフェイス内で信頼レベルやライフサイクルステータスを直接示すことができます。詳細については、ノートブックにタグを適用するをご覧ください。

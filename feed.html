AI/BI dashboard Genie integration and publish dialog improvements
> AIとBIダッシュボードのGenie統合、および公開ダイアログの改善
September 18, 2025
  • Genie credential handling for embedded dashboards: Genie integrated with dashboards published with embedded credentials now executes generated queries using the publisher's data credentials. Dashboards published without embedded credentials continue executing using the viewer's data credentials.
  • Simplified publish dialog: The dashboard publish dialog has been redesigned and simplified. The toggle for Genie integration has been moved to dashboard settings.
>
  • 組み込みダッシュボードのGenie資格情報の処理: Genieは、組み込み資格情報で公開されたダッシュボードと統合され、発行者のデータ資格情報を使用して生成されたクエリを実行します。組み込み資格情報なしで公開されたダッシュボードは、ビューアーのデータ資格情報を使用して実行を続けます。
  • 簡略化された公開ダイアログ: ダッシュボードの公開ダイアログが再設計され、簡略化されました。Genieとの統合のトグルがダッシュボードの設定に移動されました。
Genie SQL improvements, API enhancements, and bug fixes
> ジーニーSQL改善、APIの強化、およびバグ修正
September 18, 2025
  • Upgraded LLM models: The LLM models used by Genie to author SQL have been upgraded for improved accuracy and performance.
  • SQL autocomplete: Autocomplete is now available when editing Genie SQL to help users write queries more efficiently.
  • Feedback API endpoints: Thumbs up and thumbs down feedback API endpoints are now available for collecting user feedback on Genie responses.
  • Suggested follow-up questions API: Suggested follow-up questions through the API are now available.
  • Consumer entitlement API access: Users with the workspace consumer entitlement can now use the API suite.
  • Warehouse ID in API responses: GET API requests for Genie spaces now include the warehouse_id attached to the space.
>
  • アップグレードされたLLMモデル: GénieがSQL作成に使用するLLMモデルが、精度と性能の向上のためにアップグレードされました。
  • SQLオートコンプリート: GénieのSQL編集時に、ユーザーがクエリをより効率的に記述できるようオートコンプリートが利用可能になりました。
  • フィードバックAPIエンドポイント: Génieの回答に対するユーザーからのフィードバックを収集するため、「いいね」と「あまり良くない」のフィードバックAPIエンドポイントが利用可能になりました。
  • 提案されるフォローアップ質問API: APIを介して提案されるフォローアップ質問が利用可能になりました。
  • ユーザーの権限APIアクセス: ワークスペースのユーザー権限を持つユーザーがAPIを使用できるようになりました。
  • APIレスポンスのウェアハウスID: GénieのスペースのGET APIリクエストに、そのスペースに関連付けられたwarehouse_idが含まれるようになりました。
Databricks One Public Preview
> データブリックス ワン パブリック プレビュー
September 17, 2025

Databricks One, a simplified user interface designed for business users, is now in Public Preview. Databricks One provides a single, intuitive entry point to interact with data and AI in Databricks, without requiring technical knowledge of compute resources, queries, models, or notebooks.

With Databricks One, business users can:

  • View and interact with AI/BI dashboards to track KPIs and analyze metrics.
  • Ask data questions in natural language using AI/BI Genie.
  • Use custom-built Databricks Apps that combine analytics, AI, and workflows.

Workspace admins can enable Databricks One from the Previews page in the admin console.

See What is Databricks One?.

> データブリックスワン（Databricks One）は、ビジネスユーザー向けに設計された簡略化されたユーザーインターフェイスで、現在パブリックプレビューに入っています。データブリックスワンは、コンピューティングリソース、クエリ、モデル、ノートブックに関する技術的な知識なしに、データとAIをデータブリックスで直接操作できる、単一の直感的な入り口を提供します。 データブリックスワンを使うことで、ビジネスユーザーは次のことができます: - KPIをトラッキングし、メトリクスを分析するためのAI/BIダッシュボードを閲覧し、操作する。 - AI/BI Genieを使って自然言語でデータに関する質問をする。 - アナリティクス、AI、ワークフローを組み合わせたカスタムビルトのデータブリックスアプリを使う。 ワークスペースの管理者は、管理コンソールのプレビューページからデータブリックスワンを有効化できます。 詳しくは、What is Databricks One?をご覧ください。

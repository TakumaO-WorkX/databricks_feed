Dashboard enhancements and bug fixes
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011Ccf3dVu9hQhtGPqQWumxo'}]
July 02, 2026
  • Import reports from Power BI and Tableau: Authors can now import Power BI and Tableau reports as a new AI/BI dashboard directly from the dashboard list page. See Import BI files using Genie Code.
  • Improved metric view reuse when importing: When importing Tableau and Power BI reports as AI/BI dashboards, Genie Code is now more likely to reference existing Unity Catalog metric views instead of creating new ones. See Import BI files using Genie Code.
  • Hide dashboard pages before publishing: Authors can now hide dashboard pages so they aren't published, while continuing to edit them in draft mode. See Create multi-page reports.
  • STRING_AGG function: You can now use STRING_AGG to concatenate non-null string values within a group, with an optional custom delimiter. See string_agg aggregate function.
  • Custom Vega visualization improvements: Two enhancements for custom Vega-Lite visualizations. See Custom visualizations in AI/BI dashboards.
    • Cross-filtering now works for Vega-Lite charts. The selection appears in the active filter bar and is preserved in dashboard URLs and share links.
    • Custom Vega-Lite charts can now inherit the dashboard's light or dark mode and custom theme, and authors can reference theme values directly in Vega-Lite expressions.
  • Hide pivot table columns: You can now hide columns in a pivot table. See Pivot visualization.
  • Counter widget comparison styling: Counter widgets now support text styling on the comparison date and conditional coloring on the comparison field. See Counter visualization.
  • More compact filter widgets: Filter widgets on the dashboard canvas now appear more compact.
  • Dynamic table column widths: Table columns now adjust their width automatically based on cell content.
  • Apply filters fix: Fixed a bug where applying filters did not work.
  • Table column resize fix: Fixed an issue where resizing a table column could unintentionally trigger a sort on that column.
  • Combo chart hover fix: Fixed a console error that sometimes occurred when hovering over empty areas of a combo chart.

Lakeflow Connect row filtering is now generally available
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011Ccf3dXV9swSxprMMFtqTu'}]
July 02, 2026

Row filtering in Lakeflow Connect is now generally available. Row filtering allows you to ingest only the rows you need by applying conditions similar to a SQL WHERE clause, improving performance and minimizing data duplication. Row filtering applies during both the initial load and subsequent incremental updates, and is supported for SaaS connectors including Google Analytics, Salesforce, and ServiceNow, as well as all query-based connectors.

See Select rows to ingest.

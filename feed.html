Agent Bricks: Supervisor Agent is now Generally Available
> 特別エージェント・ブリックス: 監督エージェントが一般公開されました。
February 10, 2026

Agent Bricks provides a streamlined approach to operationalize data into production-grade AI agents. Use Agent Bricks: Supervisor Agent to create a multi-agent supervisor system that orchestrates AI agents and tools to work together on complex tasks.

Supervisor Agent is now Generally Available in select US regions for workspaces without Enhanced Security and Compliance features. For workspaces that have the compliance security profile enabled and HIPAA controls selected, Supervisor Agent will be Generally Available soon. See What's coming.

>

Agent Bricksは、データを本番グレードのAIエージェントにオペレーションするための効率的なアプローチを提供します。Agent Bricks: Supervisor Agentを使用して、複雑なタスクに取り組むためのマルチエージェントスーパーバイザーシステムを作成することができます。

Supervisor Agentは、Enhanced Security and Compliance機能のないワークスペースの一部のUS地域で一般提供されています。コンプライアンスセキュリティプロファイルが有効化され、HIPAA管理が選択されているワークスペースでは、Supervisor Agentは間もなく一般提供される予定です。今後の予定をご確認ください。

New documentation for authoring and deploying agents on Databricks Apps
> 新しいデータブリックスAppでのエージェントの作成とデプロイに関するドキュメント
February 10, 2026

New documentation and project templates are available for building and deploying AI agents on Databricks Apps. Author agents with popular libraries like LangGraph, PyFunc, and OpenAI Agent SDK, then deploy them on Databricks Apps. See Author an AI agent and deploy it on Databricks Apps.

>

Databricks Appsでの人工知能エージェントの構築とデプロイのための新しいドキュメントとプロジェクトテンプレートが利用可能です。LangGraph、PyFunc、OpenAI Agent SDKなどの人気ライブラリを使ってエージェントを作成し、Databricks Appsにデプロイしてください。Databricks Appsでの人工知能エージェントの作成とデプロイを参照してください。

ADBC driver is now the default driver for new Power BI connections
> ADDBCドライバーが新しい Power BI 接続のデフォルトドライバーになりました。
February 09, 2026

New connections created in Power BI Desktop or Power BI Service now automatically use the Arrow Database Connectivity (ADBC) driver by default. Existing connections continue to use ODBC unless you manually update them to ADBC. You can still switch to ODBC drivers for new connections. See Configure ADBC or ODBC driver for Power BI.

>

Power BI Desktop またはPower BI Serviceで作成された新しい接続は、デフォルトでArrow Database Connectivity (ADBC)ドライバーを使用するようになりました。既存の接続は、手動でADBCに更新しない限り、ODBCを使用し続けます。新しい接続でODBCドライバーに切り替えることもできます。 Power BIのADBCまたはODBCドライバーの設定を参照してください。

Deploy Databricks apps from Git repositories (Beta)
> Databricksアプリをギットリポジトリから展開する(ベータ版)
February 06, 2026

You can now deploy Databricks apps directly from Git repositories without uploading files to the workspace. Configure a repository for your app and deploy from any branch, tag, or commit. See Deploy from a Git repository.

>

Gitリポジトリからファイルをアップロードすることなく、Databricksアプリを直接デプロイできるようになりました。アプリのリポジトリを設定し、任意のブランチ、タグ、またはコミットからデプロイできます。Gitリポジトリからデプロイするを参照してください。

Databricks SQL version 2025.25 is rolling out in Current

Databricks SQL version 2025.25 is rolling out to the Current channel from August 20th, 2025 to August 28th, 2025. See features in 2025.25.

> データブリッククスSQL バージョン2025.25が2025年8月20日から2025年8月28日の間に現在のチャンネルにリリースされます。2025.25の機能をご確認ください。
Databricks Assistant integrated with compute
> データブリックス・アシスタント、コンピューティングと連携
August 21, 2025

Users can now chat with Databricks Assistant on some compute pages. Use the Assistant chat panel to help you create a new compute resource, pool, and policy. Or, ask the Assistant questions about the compute resource, like “Is this compute resource Unity Catalog-enabled?” or “What’s the current compute policy?”

To learn more about chatting with the Assitant, see Tour of the Assistant pane. To learn more about compute resources, see Compute.

> ユーザーは一部のコンピューティング ページで Databricks アシスタントとチャットできるようになりました。アシスタントのチャット パネルを使用して、新しいコンピューティング リソース、プール、およびポリシーの作成を支援してもらえます。また、「このコンピューティング リソースは Unity Catalog 対応ですか?」や「現在のコンピューティング ポリシーは何ですか?」などのコンピューティング リソースに関する質問をアシスタントに尋ねることができます。 アシスタントとのチャットについて詳しく学びたい場合は、「アシスタント ペインの概要」をご覧ください。コンピューティング リソースについて詳しく学びたい場合は、「コンピューティング」をご覧ください。
Dashboard sorting and formatting improvements
> ダッシュボードの並べ替えと書式設定の改善
August 21, 2025

  • Fixed sorting on temporal categorical axes: Resolved an issue where sorting did not work correctly on temporal categorical axes.

  • Numeric formatting as Bytes: Users can now format numeric values as Bytes.

  • Totals in histogram tooltips: Histograms now display totals in tooltips.

  • Axis gridlines respect custom background colors: Gridlines now automatically update based on custom color palette selections for background color.

>

  • 時間的カテゴリ軸の固定ソート: カテゴリ軸の時間的ソートが正しく機能しない問題を解決しました。

  • 数値をバイトとしてフォーマット: ユーザーは数値をバイトでフォーマットできるようになりました。

  • ヒストグラムのツールチップにトータルを表示: ヒストグラムにツールチップでトータルを表示するようになりました。

  • カスタムの背景色に合わせた軸グリッド線: グリッド線はカスタムのカラーパレットの選択に基づいて自動的に更新されます。

Genie SQL validation and space management
> SQL 検証と空間管理
August 21, 2025

  • SQL expression validation for join relationships: Added validation logic to help authors write correct SQL join relationships. If a join statement contains errors, the validation logic flags them.

  • Fixed cloning issue for spaces: Resolved a bug where cloned spaces did not correctly copy over the associated tables.

>

  • SQLの式の結合関係の検証: 正しいSQLの結合関係を作成できるように検証ロジックを追加しました。結合文に誤りがある場合、検証ロジックがそれらを示します。

  • スペースのクローニングの問題を修正: クローンしたスペースに関連するテーブルが正しくコピーされない不具合を修正しました。

Serverless private git (Public Preview)
> サーバーレスプライベートGit（パブリックプレビュー）
August 20, 2025

You can now connect a Databricks workspace to a private Git server using serverless compute and AWS PrivateLink.

See Getting Started with Databricks Serverless Private Git

>

データブリックスのワークスペースをAWSのPrivateLink経由でサーバーレスコンピューティングを使用して、プライベートなGitサーバーに接続できます。

データブリックスのサーバーレスプライベートGitの使い始めをご覧ください。

April 22, 2026

Scoped personal access tokens will be available by default for workspaces with the compliance security profile enabled in late May 2026.

Scoped personal access tokens restrict a PAT's permissions to specific API operations by assigning one or more API scopes, instead of granting the creating identity's full workspace access.

See Authenticate with Databricks personal access tokens (legacy).

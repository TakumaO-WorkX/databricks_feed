Databricks Docs Updates

Dashboard publishing, scheduling, and field management improvements
> ダッシュボードの公開、スケジューリング、およびフィールド管理の改善
October 23, 2025
  • ****Notify viewers** disabled by default for dashboard publishing**: When publishing dashboards, the Notify viewers option is now turned off by default to reduce unintended notifications. See Publish a dashboard.
  • New functions for custom calculations: Custom calculations now support over 170 functions. See Custom calculation function reference.
  • Drag-and-drop field reordering in visualizations: Authors can now drag to reorder fields when creating some visualizations such as Pivot and Line charts.
  • Waterfall visualization type: Waterfall charts are now available for dashboards. See Waterfall chart.
>
  • **デフォルトではダッシュボード公開時の閲覧者への通知が無効**: ダッシュボードを公開する際、偶発的な通知を減らすため、閲覧者に通知オプションがデフォルトで無効になりました。詳しくはダッシュボードの公開をご覧ください。
  • カスタム計算用の新しい関数: カスタム計算では170以上の関数がサポートされるようになりました。詳細はカスタム計算関数リファレンスをご覧ください。
  • ビジュアライゼーションでのフィールド並び替え機能: 作成者は、ピボットや折れ線チャートなどのビジュアライゼーションでフィールドの順序を並び替えることができます。
  • ウォーターフォールビジュアライゼーションタイプ: ウォーターフォールチャートがダッシュボードで利用できるようになりました。詳しくはウォーターフォールチャートをご覧ください。
Genie benchmark scaling and SQL function support
> ジーニー・ベンチマークのスケーリングとSQLファンクションのサポート
October 23, 2025
  • Increased benchmark limit: The maximum number of benchmarks you can add to a Genie space has been raised from 100 to 500.
  • Run benchmarks selectively: You can now select a subset of benchmark questions to start a run from either the benchmark list page or from a previous benchmark result. See Review individual evaluations.
  • Unity Catalog SQL functions as benchmark answers: You can now add Unity Catalog SQL functions as gold standard answers to benchmarks for more comprehensive testing.
>
  • ベンチマーク上限の増加: Genieスペースに追加できるベンチマークの最大数が100から500に引き上げられました。
  • 選択的にベンチマークを実行: ベンチマークリストページや前のベンチマーク結果から、ベンチマーク質問の一部を選択して実行を開始できるようになりました。個別の評価の確認を参照してください。
  • Unity Catalog SQLの関数をベンチマークの回答として使用: より包括的なテストのために、Unity Catalog SQLの関数をベンチマークの正解回答として追加できるようになりました。
Zerobus Ingest connector in Lakeflow Connect (Public Preview)
> ゼロバス・インジェストコネクタ in Lakeflow Connect (パブリックプレビュー)
October 23, 2025

The Zerobus Ingest connector in Lakeflow Connect is in Public Preview. This connector enables record-by-record data ingestion directly into Delta tables using a gRPC API. See Zerobus Ingest connector overview.

> Lakeflow Connectのゼロバスインジェストコネクターは、パブリックプレビュー中です。このコネクターは、gRPCAPIを使用して、レコード単位でデータをDeltaテーブルに直接取り込めるようにします。ゼロバスインジェストコネクターの概要については、こちらを参照してください。

Column drop behavior updated
> 列ドロップ動作が更新されました。
October 22, 2025

When you attempt to drop a column that has one or more governed tags assigned, the operation now fails. To drop a tagged column, you must first remove all governed tags from it. See Drop a column with governed tags.

>

列に1つ以上の管理されたタグが割り当てられている場合、その列を削除しようとすると、操作が失敗します。管理されたタグのついた列を削除するには、まずその列からすべての管理されたタグを削除する必要があります。管理されたタグのついた列の削除を参照してください。

Dashboard enhancements and bug fixes
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011Cd7VUZZYTRYsYdrihSRGL'}]
July 16, 2026
  • Git folders (GA): Git folders for dashboards are now generally available. See Version control dashboards with Git.
  • Run a schedule on demand: You can now select Run now on a dashboard schedule to run it immediately. See Manage schedules.
  • Dashboard variable display names: Dashboard variable drop-down options now show the column's display name instead of the field name when a display name is available.
  • Improved Ask Genie starter questions: Ask Genie suggests better starter questions for dashboards and caches them to reduce latency.
  • Genie Code error handling: When you author a dashboard with Genie Code, it now returns an error instead of retrying SQL queries that are not valid. This behavior reduces the cost of query errors.
  • Bookmark icon visibility: When viewing a published dashboard, the bookmark icon no longer appears if a dashboard has no bookmarks.
  • Ask Genie health-check fix: Ask Genie no longer shows a health-check warning when the linked Genie Agent is still usable.
  • Dashboard creation fix: Fixed a bug where some authors with the SQL access entitlement could not create or clone dashboards.
  • PNG download fix: Fixed a bug where custom fonts were dropped from the chart title and legend in single-widget PNG downloads.
  • Waterfall chart fix: Fixed a bug where y-axis formatting options on a waterfall chart did not work.

MLflow trace storage in Unity Catalog will soon be available by default for workspaces with the compliance security profile enabled
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011Cd7VUaXLkwziUNTX18UBS'}]
July 16, 2026

MLflow trace storage in Unity Catalog tables will be available by default for workspaces with the compliance security profile enabled in mid-August 2026.

Store MLflow traces in Unity Catalog tables for scalable, governed trace storage: storage is unlimited, traces are stored in OpenTelemetry (OTel) format, access is governed through Unity Catalog schema and table permissions, and you can query traces with SQL. To view traces, point an MLflow experiment at a Unity Catalog trace location.

See Store OpenTelemetry traces in Unity Catalog.

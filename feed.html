Dashboard filter and counter enhancements
> ダッシュボードのフィルターとカウンターの機能強化
November 06, 2025
  • Per-widget filter enhancements: Per-widget filters now support date ranges, relative date ranges, and number filters.
  • Counter visualization improvements: You can now use counters to track and compare metrics over time and show sparklines. To create a counter with a sparkline, select a main value and a date column. See Counter visualization.
  • Metric view metadata support: AI/BI dashboards now consume Unity Catalog metadata from metric views, including formatting and display name.
  • Horizontal chart scrollbars on hover: Scrollbars for horizontal charts now show on hover.
  • Fixed custom calculations display in draft mode: Resolved an issue where visualizations using custom calculations were not displaying in draft mode.
  • Fixed Slack notification OAuth token field: Resolved a bug where the OAuth token field appeared pre-filled with placeholder text despite no prior token being provided for dashboard subscription Slack notifications.
>
  • ウィジェットごとのフィルター機能の拡張: ウィジェットごとのフィルターは、日付範囲、相対的な日付範囲、数値フィルターをサポートするようになりました。
  • カウンター視覚化の改善: カウンターを使って時間の経過とともにメトリクスを追跡・比較し、スパークラインを表示できるようになりました。メイン値と日付列を選択することでカウンターとスパークラインを作成できます。詳細はカウンター視覚化をご覧ください。
  • メトリクスビューメタデータのサポート: AI/BIダッシュボードは、フォーマットやディスプレイ名を含むUnity Catalogのメタデータをメトリクスビューから取得するようになりました。
  • ホバー時の水平チャートスクロールバー: 水平チャートのスクロールバーがホバー時に表示されるようになりました。
  • ドラフトモードでの custom calculations の表示を修正: custom calculationsを使う視覚化がドラフトモードで表示されない問題を解決しました。
  • Slack通知のOAuthトークンフィールドを修正: ダッシュボードのサブスクリプションのSlack通知で、事前にトークンが設定されていない場合もプレースホルダーテキストが表示される問題を修正しました。
Genie visualization and query result improvements
> ジーニーのビジュアライゼーションおよびクエリ結果の改善
November 06, 2025
  • Text explanations for empty results: Genie now includes text explanations of query results when query results are empty.
  • Improved auto-generated charts: Visualizations now use query result metadata for higher-quality auto-generated charts.
  • Improved date formatting: The process for formatting date results in visualizations has been improved.
>
  • 空の結果のためのテキストの説明: Genieは、クエリ結果が空の場合、クエリ結果の文章による説明を含むようになりました。
  • 改善された自動生成されたチャート: 視覚化では、高品質の自動生成されたチャートのためにクエリ結果のメタデータを使用するようになりました。
  • 改善された日付のフォーマット: 視覚化での日付結果のフォーマット処理が改善されました。
SQL Editor visualization fix
> SQL エディターの視覚化の修正
November 06, 2025
  • Fixed tooltip display issue: Resolved an issue where tooltips were hidden behind the legend in Notebook and SQL Editor visualizations.
>
  • 固定ツールチップの表示の問題を修正: ノートブックとSQLエディターの視覚化でツールチップがレジェンドの背後に隠れる問題を解決しました。
SAP BDC connector for Databricks supports cross-cloud sharing
> SAP BDC コネクタは Databricks のクラウド間共有をサポートしています。
November 06, 2025

You can now share data between different cloud environments using the SAP Business Data Cloud (BDC) Connector for Databricks.

See Share data between SAP Business Data Cloud (BDC) and Databricks.

>

Databricksの SAP Business Data Cloud (BDC) コネクタを使用して、さまざまなクラウド環境間でデータを共有できるようになりました。

SAP Business Data Cloud (BDC) とDatabricksの間でデータを共有するをご覧ください。

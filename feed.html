List Dashboards API sort order change
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CZgdU6T4cGJNmLV4oAJDp'}]
April 06, 2026

In an upcoming release on April 6, 2026, a new version of the List Dashboards API will change the sort order of results. Dashboards will be returned in reverse chronological order by last modified date, with the most recently modified dashboard first, instead of alphabetically by title.

This is a breaking change for users who paginate results using next_page_token. Tokens generated by a previous version of the API are not valid with the new version. If you use a token from a previous version, the API returns an error:

Invalid page_token: this token was generated by a previous/different API version. Please retry without page_token.

To continue paginating after this change, start a new request without a next_page_token.

Dashboard enhancements and bug fixes
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CZgdU71oA6TKKWK2Hj5bY'}]
April 02, 2026
  • Clear applied filters individually: Dashboard viewers can now individually clear applied filters from the active selection bar.
  • Text box vertical alignment: Text box widgets now support vertical alignment (top, center, and bottom).
  • Choropleth map boundaries: Choropleth maps now support additional boundary types, including ZIP code and NUTS regions.
  • Waterfall chart total bar visibility: The waterfall chart editor now hides the total bar toggle when the x-axis configuration doesn't support it, reducing authoring confusion.
  • Genie Code faceted charts: Genie Code can now create faceted charts (also known as trellis or small multiple charts).
  • Genie Code starter questions: Genie Code starter questions are now scoped to the currently viewed dashboard page.
  • "Explain this change" chart types: The "Explain this change" feature is now available for pivot table cells, horizontal bar charts, pie charts, and heatmaps, in addition to time series charts. See Ask Genie to explain chart changes.
  • Embedding disabled error message: Fixed an error message when embedding is disabled to clarify that the workspace administrator, not the account administrator, must update the workspace setting.
  • Y-axis label alignment: Fixed an incorrect width measurement for horizontal y-axis labels that caused misaligned axis text.
  • Filter picker label truncation: Long field and parameter names in the filter picker dropdown are now truncated to prevent layout overflow.

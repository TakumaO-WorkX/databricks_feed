Anthropic Claude Haiku 4.5 now available as a Databricks-hosted model
> アンソロポック・クロード・ハイク 4.5 がデータブリックスホストのモデルとして利用可能になりました。

>

モザイクAIモデルサービングは、Databricksホスト型モデルとしてClaude Haiku 4.5をサポートしています。トークン従量課金型の基盤モデルAPIを使ってこのモデルにアクセスできます。

New Databricks accounts will not have access to legacy features
> 新しい Databricks アカウントには従来のフィーチャーにアクセスできません。
December 19, 2025

Databricks accounts created after December 18, 2025 will not have access to certain legacy features such as access to DBFS root and mounts, Hive Metastore, and No-isolation shared compute. These accounts will exclusively use Unity Catalog for unified governance and enterprise-grade security.

This behavior enforces the Disable legacy features account setting available in existing Databricks accounts. See Disable access to legacy features in new workspaces.

> 2025年12月18日以降に作成されたDatabricksアカウントは、DBFS root and mountsやHive Metastore、No-isolation shared computeなどの一部の従来のfeatureにアクセスできなくなります。これらのアカウントは、統一ガバナンスと企業レベルのセキュリティのためにUnity Catalogを専用で使用します。 既存のDatabricksアカウントで利用可能なDisable legacy featuresアカウント設定により、この動作が強制されます。詳細はDisable access to legacy features in new workspacesを参照してください。
Dashboard enhancements and fixes
> ダッシュボードの機能強化と修正
December 18, 2025
  • Boolean alias formatting: Boolean values now support alias formatting, allowing users to replace "true" and "false" with custom strings.
  • Compact global filters: Reduced the size of global filters to save space.
  • Relative date range options: Relative date range filters now include yesterday, last week, last month, and last year.
  • Filter widget background fix: Fixed an issue where a gray background appeared on some filter widgets.
  • Filter reset button fix: Fixed an issue where the "Reset to default" button appeared on the active filter bar when all filters showed default values.
  • Custom chart labels: You can now choose a custom field for chart labels. See Value labels.
  • Sortable filter dropdown options: Authors can now sort filter dropdown options by another field.
  • Multi-select filter paste: Viewers can now copy a column of values from a spreadsheet and paste them into a multi-select filter.
  • Chart tooltip overflow indicator: Chart tooltips with many series now indicate when there are more series.
  • Axis label spacing fix: Fixed an issue where long axis labels took up too much plotting area.
  • Agent-created widget consistency: Agent-created widgets now work consistently across edit and published dashboards.
  • Mark label angle fix: Fixed mark label angle for charts with multiple Y axes without an X axis.
>
  • ブール値エイリアス形式: ブール値のエイリアス形式がサポートされ、ユーザーが"true"と"false"をカスタムの文字列に置き換えることができるようになりました。
  • コンパクトなグローバルフィルター: グローバルフィルターのサイズを縮小し、スペースを節約しました。
  • 相対日付範囲オプション: 相対日付範囲フィルターに、昨日、先週、先月、および昨年が含まれるようになりました。
  • フィルターウィジェットの背景の修正: 一部のフィルターウィジェットにグレーの背景が表示される問題を修正しました。
  • フィルターリセットボタンの修正: すべてのフィルターがデフォルト値を表示しているときに、アクティブなフィルターバーに"デフォルトにリセット"ボタンが表示される問題を修正しました。
  • カスタムチャートラベル: チャートラベルのためのカスタムフィールドを選択できるようになりました。値ラベルを参照してください。
  • ソート可能なフィルタードロップダウンオプション: 作成者は、他のフィールドによってフィルターのドロップダウンオプションをソートできるようになりました。
  • マルチ選択フィルターの貼り付け: 閲覧者は、スプレッドシートから値のカラムをコピーし、マルチ選択フィルターに貼り付けることができます。
  • チャートツールチップのオーバーフロー表示: 多くのシリーズを持つチャートツールチップに、さらにシリーズがあることを示すインジケーターが追加されました。
  • 軸ラベルの間隔の修正: 長い軸ラベルが描画領域を過剰に占める問題を修正しました。
  • エージェント作成ウィジェットの一貫性: エージェント作成ウィジェットが、編集中のダッシュボードと公開されたダッシュボードで一貫して動作するようになりました。
  • マークラベルの角度の修正: X軸のないチャートで、複数のY軸を持つチャートのマークラベルの角度を修正しました。

Public URLs for ODBC driver downloads will be disabled
> パブリックURLのODBCドライバーダウンロードは無効化されます。
December 18, 2025

In an upcoming release, public URLs for automated Apache Spark ODBC driver downloads will be disabled. After this change, all driver downloads, including automated processes, will require authentication. This will break automated processes that download Apache Spark ODBC drivers using direct public links without authentication.

> 自動化されたApache Spark ODBCドライバーのダウンロード用の一般向けURLが、今後のリリースで無効化されます。この変更後、認証なしで直接公開リンクを使ったダウンロードが含め、すべてのドライバーダウンロードには認証が必要になります。これにより、認証なしでApache Spark ODBCドライバーをダウンロードする自動化プロセスが破綻します。

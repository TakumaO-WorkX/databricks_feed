Change default workspace access to Consumer access (Public Preview)
> コンシューマーアクセス（パブリックプレビュー）へのデフォルトのワークスペースアクセスの変更
January 06, 2026

Workspace admins can now configure new users to receive Consumer access by default using group cloning. Consumer access provides access to Databricks One, a view-only experience for business users who need to view and interact with dashboards, Genie spaces, and apps without creating workspace objects. The process creates a new group for existing users and updates the users group to grant only Consumer entitlement to new users by default. This preserves existing users' authoring privileges while streamlining consumer onboarding at scale. See Change default workspace access to Consumer access.

> ワークスペースの管理者は、グループのクローン作成を使って、新規ユーザーをデフォルトでコンシューマーアクセスを持つように設定できます。コンシューマーアクセスには、ダッシュボード、Genieスペース、アプリケーションの閲覧と操作が可能な、ビューオンリーの「Databricks One」が含まれます。このプロセスでは、既存ユーザーのために新しいグループが作成され、「ユーザー」グループが更新されて、新規ユーザーにはデフォルトでコンシューマーの権限のみが付与されます。これにより、既存ユーザーの執筆権限が保たれつつ、大規模なコンシューマーのオンボーディングが簡素化されます。詳細は「[デフォルトのワークスペースアクセスをコンシューマーアクセスに変更する](https://docs.databricks.com/aws/en/security/auth/change-default-workspace-access)」をご覧ください。
Create skills for Databricks Assistant
> データブリックス・アシスタントのスキルを作成する
January 06, 2026

You can now create skills to extend Databricks Assistant in agent mode with specialized capabilities for domain-specific tasks. User skills follow the open Agent Skills standard and are automatically loaded when relevant.

See Extend the Assistant with agent skills.

>

データブリックス アシスタントをエージェントモードで拡張し、ドメイン固有のタスクのための特殊な機能を持つスキルを作成できるようになりました。ユーザースキルはオープンなAgent Skills標準に従い、関連する場合に自動的にロードされます。

エージェントスキルでアシスタントを拡張するをご覧ください。

Databricks Assistant Agent Mode is now in Public Preview
> データブリックス・アシスタント・エージェントモードが公開プレビューに入りました。
December 23, 2025

The Databricks Assistant Agent Mode preview is now enabled by default for most customers.

  • The agent can automate multiple steps. From a single prompt, it can retrieve relevant assets, generate and run code, fix errors automatically, and visualize results. It adds the ability to sample data and cell outputs to provide better results.
  • The Assistant in Agent Mode will choose between Azure OpenAI or Anthropic on Databricks (uses endpoints hosted by Databricks Inc. in AWS within the Databricks security perimeter), and is only available when the partner-powered AI features setting is enabled.
  • Admins can disable the preview if needed until the feature reaches General Availability.

See Use the Data Science Agent, the blog post, and Partner-powered AI features.

>

Databricksアシスタントエージェントモードのプレビューがほとんどのお客様のデフォルトで有効になっています。

  • エージェントは複数のステップを自動化できます。単一のプロンプトから、関連するアセットを取得し、コードを生成・実行し、エラーを自動的に修正し、結果を視覚化できます。データのサンプリングとセル出力を追加することで、より良い結果を提供できます。
  • エージェントモードのアシスタントは、Azure OpenAIまたはAnthropicのいずれかをデータブリックスで選択します(Databricks Inc.がAWS内のデータブリックスのセキュリティ境界でホストするエンドポイントを使用)。パートナー提供のAI機能設定が有効になっている場合にのみ利用できます。
  • 管理者は、機能が一般利用可能になるまでプレビューを無効にできます。

データサイエンスエージェントの使用、ブログ記事、およびパートナー提供のAI機能を参照してください。

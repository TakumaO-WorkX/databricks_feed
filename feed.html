Databricks Docs Updates

5X-Large SQL warehouse size (Beta)
> 5X大型SQLウェアハウスサイズ(ベータ)
March 13, 2026

The 5X-Large cluster size for serverless and pro SQL warehouses is now available in Beta. For serverless SQL warehouses, 5X-Large is available in all supported regions. Workspace admins can control access to this feature from the Previews page. See Manage Databricks previews.

For more information on SQL warehouse sizing, see SQL warehouse sizing, scaling, and queuing behavior.

>

サーバーレスおよびProSQL倉庫のための5X-Largeクラスターサイズが、ベータ版で利用可能になりました。サーバーレスSQL倉庫の場合、5X-Largeは全ての対応リージョンで利用可能です。ワークスペース管理者は、プレビューページからこの機能へのアクセスを制御できます。Databricksのプレビューを管理するを参照してください。

SQL倉庫のサイジングの詳細については、SQL倉庫のサイジング、スケーリング、およびキューイング動作を参照してください。

Dashboard enhancements and improvements
> ダッシュボードの機能強化と改善
March 12, 2026
  • Dashboard embedding for external users is generally available: Embed AI/BI dashboards in external applications without requiring viewers to have Databricks accounts. See What is embedding for external users?.
  • Hide Databricks logo in embedded dashboards: Externally embedded dashboards now support hiding the Databricks logo using the hideDatabricksLogo option. See Hide the Databricks logo.
  • Metric view lineage highlighting: Hovering over metric views in the lineage panel now highlights the visualizations built on those metric views.
  • Certification and deprecation icons in lineage: Data objects in the dashboard lineage panel now show their certification or deprecation status.
  • Clear individual active filters: You can now remove a single filter selection from the active filter bar without resetting all active filters.
  • Mobile layouts: Dashboards now automatically render in a mobile-friendly layout when viewed on small screens. See Mobile layout.
  • Custom subscription email subjects: When configuring a dashboard subscription, authors can now edit the subject line of notification emails sent to subscribers.
  • Accounting-style number formatting: Number fields now support accounting-style formatting, which displays negative values in parentheses and vertically aligns currency symbols.
  • Custom fonts: Dashboard authors can now select custom fonts for dashboards. Fonts must be preinstalled on viewer machines. If a font is not available, dashboards fall back to Databricks default fonts. See Set a custom font.
  • Genie Code for dashboard authoring (Public Preview): Automate multi-step dashboard workflows using natural language prompts to create datasets, visualizations, layouts, and filters. See Use Genie Code for dashboard authoring.
>

  • ライネージの認定およびデプリケーションアイコン: ダッシュボードのライネージパネルにある各データオブジェクトに、認定またはデプリケーションのステータスが表示されるようになりました。
  • 個別のアクティブフィルターのクリア: アクティブフィルターバーから、単一のフィルター選択を削除できるようになりました。他のアクティブフィルターはリセットされません。

Genie enhancements and improvements
> ジーニーの強化と改善
March 12, 2026
  • Workspace-level color palettes: Charts and graphs generated in Genie now use the visualization color palette configured at the workspace level.
  • Genie space management APIs are generally available: The CreateSpace, UpdateSpace, GenieGetSpace, GenieListSpaces, and GenieTrashSpace APIs hare now general availability. See Use the Genie API to integrate Genie into your applications.
  • API instruction length fix: Resolved a bug with the maximum instruction length in the CreateSpace and UpdateSpace APIs.
  • Improved context handling: Genie now better identifies relevant previous messages to include in context, reducing context loss.
  • Improved top-X answers with ties: Genie now more accurately answers top-X questions when results include ties.
>
  • ワークスペースレベルのカラーパレット:Genieで生成されたチャートやグラフは、ワークスペースレベルで設定された視覚化カラーパレットを使用しています。
  • Genieスペース管理APIが一般公開されました:CreateSpace、UpdateSpace、GenieGetSpace、GenieListSpaces、GenieTrashSpace APIが一般公開されました。Use the Genie API to integrate Genie into your applicationsを参照してください。
  • API命令長の修正:CreateSpaceおよびUpdateSpaceAPIの最大命令長のバグが修正されました。
  • コンテキストの処理改善:Genieは、コンテキストの損失を減らすため、関連する過去のメッセージをより正確に識別するようになりました。
  • Top-X回答の改善(同点):Genieは、結果に同点が含まれる場合でも、Top-X質問により正確に回答するようになりました。
Account-level Databricks One (Beta)
> アカウントレベルのDatabricks One (ベータ版)
March 12, 2026
  • Account-level Databricks One (Beta): Users can now discover and interact with AI/BI dashboards, Genie spaces, and Databricks Apps from all workspaces in their account from a single entry point. Account-level Databricks One provides a unified homepage, cross-account search, and recommendations and favorites across workspaces. See Account-level Databricks One.
>
  • アカウントレベルの Databricks One (ベータ版): ユーザーは、自身のアカウントにあるすべてのワークスペースから、AI/BI ダッシュボード、Genie スペース、およびDatabricks アプリを検索して操作できるようになりました。アカウントレベルの Databricks One では、統一されたホームページ、クロスアカウントの検索、ワークスペース間の推奨機能とお気に入りの設定が利用できます。詳細は アカウントレベルの Databricks One をご覧ください。

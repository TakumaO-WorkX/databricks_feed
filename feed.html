Public URLs for JDBC driver downloads will be disabled
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CdLmTBnzHZZxY1PuVS7q3'}]
July 24, 2026

Beginning September 2026, public S3 links for downloading the Simba JDBC Driver (Legacy) will be disabled. After this date, scripts or processes that use direct public links to download the driver will no longer function. In addition, downloading from the JDBC archive page will require signing in to Databricks.

To continue using automated downloads, migrate to the Databricks JDBC Driver before September 2026. See Databricks JDBC Driver.

Dashboard enhancements and bug fixes
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CdLmTEWwwnqSB43pkCrdd'}]
July 23, 2026

  • Import BI files (Public Preview): Importing Power BI and Tableau reports as AI/BI dashboards with Genie Code is in Public Preview. See Import BI files using Genie Code.

  • Image widget: You can now add a dedicated image widget to a dashboard. See Image widgets.

  • Multi-select filter improvements: Multi-select filter improvements:

    • Multi-select filter stays open: Multi-select filter drop-downs now stay open between selections and keep your place in the list instead of closing.
    • Select all matching values in a multi-select filter: In a multi-select filter, you can now type to narrow the list and select Select all results to select every matching value at once. Selections are additive across searches.

  • Dashboard variable display names: When a display name is available, dashboard variable drop-down menu options show the column's display name instead of the field name. To learn more about variables, see Create a dashboard variable.

  • Cross-filtering text-selection fix: Fixed a bug where selecting text in a table unintentionally triggered cross-filtering.

  • Genie Code for dashboard authoring (GA): Genie Code for dashboard authoring, the Genie agent built for authoring dashboards, is now generally available. See Use Genie Code for dashboard authoring. This release also adds the following authoring capabilities:

    • Author image widgets with Genie Code: Genie Code for dashboard authoring can now add image widgets to a dashboard. See Author image widgets.
    • Beautify dashboards with Genie Code: Genie Code for dashboard authoring can now redesign a dashboard's appearance. Ask it to beautify your dashboard, then undo the changes if you want to revert. See Beautify a dashboard.

The Lakeflow Pipelines Editor has been generally available since May 2026. In August, it will become the default editor for developing and debugging pipelines in workspaces with the compliance security profile enabled. At that time the legacy editor will be removed.

For more information about the Lakeflow Pipelines Editor, see Develop and debug ETL pipelines with the Lakeflow Pipelines Editor. If you have questions about this transition, contact your account team.

Dashboard enhancements and bug fixes
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CctKjWPFmwhYE19xQGkeK'}]
July 09, 2026
  • Import BI files is generally available: Importing Power BI and Tableau reports as AI/BI dashboards with Genie Code is now generally available. You can also start an import directly from the dashboard list page. See Import BI files using Genie Code.
  • Dashboard Ask Genie is generally available: The Ask Genie button on published dashboards is now generally available. See Genie Agents with dashboards.
  • Ask Genie handles partially supported datasets: When only some datasets in a dashboard use unsupported SQL expressions, Ask Genie now answers from the supported datasets instead of failing. See Incompatible SQL expressions.
  • English names for choropleth regions: Choropleth maps now support English region names from the name_en column as the identifier for admin-0 (country) boundaries. See Region.
  • Hide pivot table column headers: Authors can now hide the column header in a pivot table. See Value-level settings.
  • Genie Code on by default for compliance security profile: For workspaces with a compliance security profile (CSP), Genie Code for dashboard authoring is now turned on by default and runs on Databricks-hosted models. See Use Genie Code for dashboard authoring.
  • Empty multi-select filter defaults in Genie Code: Genie Code can now use an empty multi-select ("All") default for parameter filters instead of requiring a single default value.
  • Cluster-by and partition-by for local metric views: Local metric views now support cluster-by and partition-by configuration in the UI and YAML. See Local metric views.
  • Apply filters fix: Fixed a bug where the apply button on filters cleared the text input instead of applying the filter.
  • Dashboard import fix: Fixed a bug where importing a dashboard failed if it contained a bookmark that referenced a deleted widget.

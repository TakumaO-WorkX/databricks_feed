Dashboard pivot tables, filters, and data export improvements
> ダッシュボードのピボットテーブル、フィルター、およびデータ出力の改善
January 08, 2026
  • JSON cell copy-paste improvement: JSON objects in table cells now remain expanded when you drag to copy-paste.
  • Parameterized default catalog and schemas: Parameterize the default catalog and schemas when deploying dashboards using Databricks Asset Bundles. See dashboard.
  • Multi-select filter paste enhancement: Paste comma-separated and newline-separated values into multi-select filters.
  • Pivot table total rows: Pivot tables can now display total rows at the top.
  • Dashboard widget drag handles: Dashboard widgets now have drag handles for resizing and repositioning.
  • Pivoted table Excel downloads: Download dashboard data to Excel in pivoted format rather than flattened. See Download pivot tables to Excel.
  • Timezone display fixes: Fixed a number of timezone-related issues, ensuring that displayed timezones match underlying data timezones instead of displaying UTC.
  • Selective data point labels: Labels can now be shown only for first/last or min/max data points.
>
  • JSONセルのコピー＆ペースト機能の改善:テーブルセル内のJSON オブジェクトをドラッグしてコピー＆ペーストすると、展開されたままの状態を維持できるようになりました。
  • パラメータ化された既定のカタログとスキーマ: Databricksのアセットバンドルを使ってダッシュボードをデプロイする際に、既定のカタログとスキーマをパラメータ化できるようになりました。詳細はダッシュボードのドキュメントをご覧ください。
  • マルチ選択フィルターのペースト機能の強化: コンマ区切りやNewline区切りの値をマルチ選択フィルターにペーストできるようになりました。
  • ピボットテーブルの合計行:ピボットテーブルに合計行を表示できるようになりました。
  • ダッシュボードウィジェットのドラッグハンドル: ダッシュボードウィジェットにサイズ変更や位置の変更ができるドラッグハンドルが追加されました。
  • ピボットされたテーブルのExcel出力: ダッシュボードデータをフラット形式ではなく、ピボット形式でExcelにダウンロードできるようになりました。詳細はピボットテーブルのExcelダウンロードをご覧ください。
  • タイムゾーンの表示修正: 表示されるタイムゾーンが、基になるデータのタイムゾーンではなくUTCになっていた問題を修正しました。
  • 選択的なデータポイントラベル: 最初/最後、または最小/最大のデータポイントのみラベルを表示できるようになりました。
Genie PDF export, citations, and accuracy improvements
> PDF書き出し、引用、および正確性の向上
January 08, 2026
  • PDF export for Research Agent (Beta) reports: Export Genie Research Agent reports as PDF files. See Export Research Agent reports.
  • SQL example citations: Genie's analysis steps now cite the SQL examples used. See Review a response.
  • Currency formatting improvements: Numeric columns in table results now use more conservative currency formatting rules to avoid incorrect assumptions.
  • Improved question type handling: Genie now categorizes common question types (top K, percentage, and distribution) more accurately so the right context is applied, improving results.
>
  • Research Agent (Beta)レポートのPDFエクスポート: Genie Research AgentレポートをPDF形式でエクスポートします。Research Agentレポートのエクスポートを参照してください。
  • SQLの例の引用: Genieの分析手順では、使用されたSQLの例が引用されるようになりました。レスポンスの確認を参照してください。
  • 通貨フォーマットの改善: テーブル結果の数値列では、誤った想定を避けるために、より慎重な通貨フォーマットのルールが使用されるようになりました。
  • 質問タイプの処理の改善: Genieは、トップK、パーセンテージ、分布などの一般的な質問タイプをより正確に分類するようになり、適切なコンテキストが適用されるようになり、結果が改善されました。
Genie value sampling features renamed
> ジーニーの価値サンプリング機能が名称変更されました。
January 07, 2026

The value sampling features in Genie will soon be renamed to better describe their function:

  • Value sampling will be renamed to prompt matching
  • Example values will be renamed to format assistance
  • Value dictionaries will be renamed to entity matching
> ジーニーの値のサンプリング機能は、その機能をよりよく説明するために間もなく名称が変更されます: - 値のサンプリング(Value sampling)は、プロンプトマッチング(prompt matching)に名称が変更されます。 - 例示値(Example values)は、フォーマットアシスタンス(format assistance)に名称が変更されます。 - 値辞書(Value dictionaries)は、エンティティマッチング(entity matching)に名称が変更されます。

Databricks Docs Updates

Value sampling features renamed
> 価値サンプリング機能が改称されました。
January 16, 2026

The value sampling features in Genie have been renamed to better describe their function:

  • Prompt matching (previously value sampling)
  • Format assistance (previously example values)
  • Entity matching (previously value dictionaries)
>

Genieの値サンプリング機能は、その機能をより適切に説明するために名称が変更されました:

  • プロンプトマッチング (以前の値サンプリング)
  • フォーマット支援 (以前の例値)
  • エンティティマッチング (以前の値辞書)
Agent Bricks: Knowledge Assistant will soon be available by default for workspaces with enhanced security and compliance
> エージェントブリックス:ナレッジアシスタントは、強化されたセキュリティとコンプライアンスを備えたワークスペースでデフォルトで間もなく利用可能になります。
January 15, 2026

In mid-February 2026, Agent Bricks: Knowledge Assistant will be available by default for workspaces that have Enhanced Security and Compliance enabled.

Agent Bricks provides a streamlined approach to operationalize data into production-grade AI agents. Use Agent Bricks: Knowledge Assistant to create a chatbot that can answer questions about your documents and provide high-quality responses with citations.

Users can start using this feature now by enabling the Knowledge Assistant preview.

> 2026年2月中旬、Agent Bricks: Knowledge Assistantは、Enhanced Security and Complianceが有効になっているワークスペースのデフォルトで利用可能になります。 Agent Bricksは、データを本番品質のAIエージェントに確実に運用するための効率的なアプローチを提供します。Agent Bricks: Knowledge Assistantを使用して、ドキュメントに関する質問に回答し、引用を含む高品質な回答を提供するチャットボットを作成できます。 ユーザーは今すぐこの機能を使い始めることができ、Knowledge Assistantプレビューを有効にするだけです。
ADBC will become the default driver for new Power BI connections
> ADPCはPower BIの新しい接続のデフォルトドライバーになる
January 15, 2026

Starting in February, new connections created in Power BI Desktop or Power BI Service will automatically use the Arrow Database Connectivity (ADBC) driver by default. Existing connections will continue to use ODBC unless you manually update them to ADBC.

If you prefer to continue using ODBC for new connections, you will have two options:

  1. When creating a connection, select default as the Implementation.
  2. In Power BI Desktop, disable the ADBC feature preview, which will restore ODBC as the default for all new connections.

To learn more about ADBC, see Arrow Database Connectivity (ADBC) driver for Power BI and the Databricks Driver in the ADBC repository.

> 2月から、Power BI Desktop またはPower BI Serviceで作成された新しい接続は、デフォルトでArrow Database Connectivity (ADBC)ドライバーを使用するようになります。既存の接続は、手動で更新しない限り、ODBCを継続して使用します。 新しい接続でODBCを引き続き使用したい場合は、2つのオプションがあります。 1. 接続を作成する際に、**実装**としてdefaultを選択してください。 2. Power BI Desktopで、ADBC機能プレビューを無効にすると、すべての新しい接続でODBCがデフォルトに戻ります。 ADBCの詳細については、「Power BIのArrow Database Connectivity (ADBC)ドライバー」および「ADBCリポジトリのDatabricksドライバー」を参照してください。
Dashboard navigation, pivot tables, and text widget improvements
> ダッシュボードナビゲーション、ピボットテーブル、およびテキストウィジェットの改善
January 15, 2026
  • Dashboard page navigation scroll bar: The horizontal scroll bar to navigate between dashboard pages is now thicker.
  • Dashboard title character limit: Dashboard titles now have more horizontal space before truncating.
  • Pivot table header height: Pivot table header height can be configured to enable wrapping for multi-row heights.
  • Text widget markdown tables: Text widgets now support markdown table syntax.
>
  • ダッシュボードページのナビゲーションスクロールバー: ダッシュボードページ間を移動するための水平スクロールバーが太くなりました。
  • ダッシュボードタイトルの文字数制限: ダッシュボードタイトルの表示スペースが増えたため、文字が切れなくなりました。
  • ピボットテーブルのヘッダー高さ: ピボットテーブルのヘッダー高さを調整して、複数行に折り返すことができるようになりました。
  • テキストウィジェットのMarkdownテーブル: テキストウィジェットでMarkdownのテーブル構文がサポートされるようになりました。

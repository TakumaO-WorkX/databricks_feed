Dashboard enhancements and bug fixes
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CapwBLpsuL6haraLB53QY'}]
May 07, 2026
  • Improved error messages: Dashboard error messages can now be more easily viewed and copied.
  • Reset draft API: An API is now available for dashboard authors to reset the draft to the last published version.
  • Undo-redo on the data page: Dashboards in draft mode now support undo and redo on the data page.
  • Y-axis fix: Fixed a bug where the Y-axis of faceted bar and area charts started at -0.5 instead of 0.
  • Multi-select filter fix: Fixed a usability issue where multi-select filters always scrolled to the top of the list after a selection.
  • Local metric view Hive Metastore fix (Public Preview): Fixed an issue causing local metric views to fail when referencing Hive Metastore tables rather than Unity Catalog tables.
  • Local metric view preview fix (Public Preview): Fixed an issue where preview charts for measures and dimensions defined in a local metric view were not rendering in the data tab.
  • Metric view SQL editor improvements (Public Preview): Several UX improvements to the metric view SQL editor, including taller SQL and expression editors in the side panel, layout padding improvements, and a cropping fix in the join modal table source selector.
  • Local metric view join autofill (Public Preview): Local metric views now autopopulate join conditions when joining two or more tables.

Faster package installs with %uv pip in serverless notebooks
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CapwBQAHuKVEYpTvU7hZd'}]
May 07, 2026

You can now use the %uv pip magic command in serverless notebooks on environment version 5 or above as a faster alternative to %pip for Python package management. %uv pip is powered by the uv package manager and shares the same notebook virtual environment as %pip.

See Faster installs with %uv pip.

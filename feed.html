Databricks Docs Updates

Governed Tags will be generally available in mid-March
> 3月中旬に一般公開される予定のガバナードタグ
February 26, 2026

In mid-March 2026, Governed Tags will be generally available. Governed Tags allow administrators to define a controlled set of tags that can be applied to Unity Catalog objects, helping organizations enforce consistent metadata tagging across their lakehouse.

> 2026年3月中旬に、Governed Tagsが一般提供されます。Governed Tagsにより、管理者はUnity Catalogオブジェクトに適用できる管理されたタグセットを定義できるようになり、組織全体でデータレイクのメタデータタグ付けの一貫性を強化できます。
Sample Data Explorer will be generally available in late March
> サンプル データ エクスプローラーは、3月下旬に一般提供されます。
February 26, 2026

In late March 2026, Sample Data Explorer with Databricks Assistant will be generally available. Sample Data Explorer lets you ask questions about tables in Unity Catalog using an embedded natural language interface.

>

2026年3月下旬、Databricks AssistantでSample Data Explorerが一般公開されます。Sample Data Explorerを使えば、Unity Catalogのテーブルについて自然言語インターフェースを使って質問できます。

Dashboard enhancements and improvements
> ダッシュボードの機能拡張と改善
February 26, 2026
  • Unity Catalog metric view lineage: Dashboard info now shows Unity Catalog metric views in the lineage section.
  • Dashboard refresh cancellation: Cancelling a dashboard refresh now shows the last available data instead of an error.
  • DD-MM-YYYY date format: The DD-MM-YYYY date format is now available for date fields.
  • Date range parameter "All" selection: Date range parameters now support All. When All is selected, the parameter value is null and all rows are returned. See Apply date range parameters.
  • Genie agentic experience (Public Preview): Genie for dashboard insights now offers an agentic experience (Public Preview) by default.
  • Point map recentering: Point maps now recenter upon relevant points when a filter is applied.
  • Datetime drop-down menu fix: Fixed an issue where datetime drop-down menus could not be set in the global filters panel.
>
  • Unity Catalogメトリックビューのlineage: ダッシュボードの情報に、Unity Catalogメトリックビューがlineageセクションに表示されるようになりました。
  • ダッシュボードリフレッシュのキャンセル: ダッシュボードリフレッシュをキャンセルすると、エラーではなく最新のデータが表示されるようになりました。
  • DD-MM-YYYY日付形式: DD-MM-YYYY日付形式が日付フィールドで使用可能になりました。
  • 日付範囲パラメーター「全て」の選択: 日付範囲パラメーターに「全て」が追加されました。「全て」を選択すると、パラメーター値がnullになり、すべての行が返されます。日付範囲パラメーターの適用を参照してください。
  • Genieエージェント体験(パブリックプレビュー): ダッシュボードのインサイトにおけるGenieがデフォルトでエージェント体験(パブリックプレビュー)を提供するようになりました。
  • ポイントマップの再中心化: フィルターが適用されると、ポイントマップが関連するポイントを中心に再中心化されるようになりました。
  • 日時ドロップダウンメニューの修正: グローバルフィルターパネルで日時ドロップダウンメニューを設定できない問題が修正されました。
Genie enhancements and improvements
> 天性の向上および改善
February 26, 2026
  • Metric view display names: Genie now uses metric view display names and display formats for query result tables.
  • Automatic JOIN relationships: Creating and updating Genie spaces using the API now automatically creates JOIN relationships based on the primary and foreign keys defined in source tables.
  • Context tab for space consumers: A new Context tab is now available for Genie space consumers with CAN VIEW or CAN RUN permissions. Consumers can view the space description and common questions at any point during their Genie interactions. See Use a Genie space to explore business data.
  • Agent mode (Beta) thinking traces: Agent mode now streams thinking traces in-line for improved readability.
  • Agent mode (Beta) parallel queries: Agent mode now runs queries in parallel for faster results.
>
  • メトリックビューの表示名: Genieは、クエリ結果テーブルのメトリックビューの表示名と表示形式を使用するようになりました。
  • 自動JOIN関係: APIを使用してGenieスペースを作成および更新すると、ソーステーブルで定義されたプライマリキーと外部キーに基づいて自動的にJOIN関係が作成されます。
  • スペースコンシューマーのためのコンテキストタブ: CAN VIEWまたはCAN RUNアクセス許可を持つGenieスペースコンシューマーには、新しいコンテキストタブが利用できるようになりました。コンシューマーは、Genieインタラクションの任意の時点でスペースの説明と一般的な質問を確認できます。詳細はGenieスペースを使用してビジネスデータを探索するを参照してください。
  • エージェントモード (ベータ) の思考過程の表示: エージェントモードでは、可読性が向上するよう、思考過程が直接表示されるようになりました。
  • エージェントモード (ベータ) の並列クエリ: エージェントモードでは、より速い結果を得るため、クエリが並列に実行されるようになりました。
ISMAP compliance support
> ISMAP準拠サポート
February 26, 2026

You can now configure Databricks workspaces to support ISMAP, the Japanese government certification initiative for cloud services. See Information system Security Management and Assessment Program (ISMAP).

> あなたは今、データブリックスのワークスペースを設定して、クラウドサービスのための日本政府の認証制度であるISMAPをサポートできます。「Information system Security Management and Assessment Program (ISMAP)」をご覧ください。

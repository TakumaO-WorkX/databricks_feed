Dashboard search and embedding enhancements

November 13, 2025
  • Enhanced search functionality: Users can now search over dashboard names, page names, widget titles and descriptions, dataset names, and their queries. Dataset queries are searchable only from the workspace search bar.

  • ブラウザのウィンドウ遷移の不具合を修正: ダッシュボードを表示中に別のブラウザウィンドウに遷移してから戻ると、ダッシュボードにデータが表示されないバグを修正しました。

Convert foreign tables to Unity Catalog managed or external tables (Public Preview)
> Unity Catalog 管理または外部テーブルへの外部テーブルの変換 (パブリックプレビュー)
November 13, 2025

You can now convert foreign tables from external catalogs (such as Hive Metastore or AWS Glue) to Unity Catalog managed or external tables while retaining table history and configurations. Use ALTER TABLE SET MANAGED to convert to managed tables with MOVE (disable source access) or COPY (keep source accessible) options, or use ALTER TABLE SET EXTERNAL to convert to external tables with an optional DRY RUN validation. Additionally, you can convert entire foreign catalogs to standard Unity Catalog catalogs using ALTER CATALOG DROP CONNECTION. See Convert a foreign table to a managed Unity Catalog table and Convert a foreign table to an external Unity Catalog table.

Minimum Databricks Runtime versions:

  • ALTER TABLE SET MANAGED: DBR 17.3 LTS or above
  • ALTER TABLE SET EXTERNAL: DBR 17.0 LTS or above
  • ALTER CATALOG DROP CONNECTION: DBR 17.3 LTS or above
  • ALTER SCHEMA SET MANAGED LOCATION: DBR 16.4 LTS or above
> 外部カタログ（Hive Metastoreや AWS Glue など）から Unity Catalog管理テーブルや外部テーブルへのテーブル変換が可能になりました。ALTER TABLE SET MANAGEDコマンドを使ってテーブルを管理テーブルに変換できます。MOVE（ソースアクセスを無効化）やCOPY（ソースアクセスを維持）オプションを指定できます。また、ALTER TABLE SET EXTERNALコマンドを使って外部テーブルに変換できます。DRY RUN検証も可能です。さらに、ALTER CATALOG DROP CONNECTIONコマンドを使って、外部カタログ全体をUnity Catalogのスタンダードカタログに変換できます。 最低限必要なデータブリックスランタイムバージョンは以下の通りです: - ALTER TABLE SET MANAGED: DBR 17.3 LTS以上 - ALTER TABLE SET EXTERNAL: DBR 17.0 LTS以上 - ALTER CATALOG DROP CONNECTION: DBR 17.3 LTS以上 - ALTER SCHEMA SET MANAGED LOCATION: DBR 16.4 LTS以上

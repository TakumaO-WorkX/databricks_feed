Databricks Docs Updates

Databricks Connector for Google Sheets offers additional features (Public Preview)
> データブリックスの Google Sheets コネクタには、追加の機能(パブリックプレビュー)があります。
October 17, 2025

The Databricks Connector for Google Sheets introduces improved query management options. Users can save queries within a Google Sheet spreadsheet, enabling easy data refresh, query reuse, and query editing.

See Connect to Databricks from Google Sheets.

> Databricksのグーグルシート用コネクタは、クエリ管理オプションを改善しました。ユーザーはグーグルシートスプレッドシート内でクエリを保存できるようになり、簡単なデータ更新、クエリの再利用、クエリの編集が可能になりました。 グーグルシートからDatabricksに接続するにはこちらをご覧ください。
Dashboard pivot tables, gridlines, and drill-through enhancements
> ダッシュボードのピボットテーブル、グリッドライン、およびドリルスルー機能の強化
October 16, 2025
  • Dynamic text functions in pivot tables: Pivot tables now support contains(), startswith(), and endswith() functions for conditional formatting. See Conditional formatting with text functions.
  • Gridline toggle: Charts now support toggling gridlines on and off. See Gridlines
  • Dashboard tags and certification in Public Preview: You can now add tags to dashboards and Genie spaces to improve organization, search, and discovery across your workspace. Tagged items are indexed for search, making it easier for users to find relevant dashboards and spaces quickly. See Manage dashboard tags and Add tags.
  • Drill-through without target filters: Drill-through now filters any visualization based on the same dataset as the source selection without an explicit target filter. If a target filter exists, it is still applied. See Use drill through to filter data.
  • Fixed discarded dashboard API error: APIs now return 404 NOT_FOUND instead of 403 PERMISSION_DENIED when accessing dashboards in the trash.
>
  • ピボットテーブルの動的テキスト関数: ピボットテーブルでは、条件付き書式設定に contains()、startswith()、endswith() 関数がサポートされるようになりました。テキスト関数を使った条件付き書式設定を参照してください。
  • グリッド線の切り替え: チャートでグリッド線の表示/非表示を切り替えられるようになりました。グリッド線を参照してください。
  • ダッシュボードのタグ付けと認証がパブリックプレビューに: ダッシュボードとGenieスペースにタグを付けることで、ワークスペース全体の整理、検索、発見が容易になります。タグ付きのアイテムはインデックス化され、関連のダッシュボードやスペースをすぐに見つけられます。ダッシュボードのタグ管理とタグの追加を参照してください。
  • ターゲットフィルターなしのドリルスルー: ドリルスルーでは、ソースの選択と同じデータセットを持つ任意の視覚化がフィルタリングされます。ターゲットフィルターが存在する場合は、それも適用されます。ドリルスルーを使ってデータをフィルタリングするを参照してください。
  • 破棄されたダッシュボードのAPI エラーを修正: ごみ箱のダッシュボードにアクセスした場合、API は 403 PERMISSION_DENIED ではなく 404 NOT_FOUND を返すようになりました。
Genie API updates, benchmark explanations, and JOIN improvements
> APIアップデート、ベンチマーク説明、JOINの改善
October 16, 2025
  • JOIN relationship discovery improvements: You can now sort JOIN relationships and SQL snippet definitions for easier management.
  • API endpoints moving to Public Preview: The following Genie API endpoints are moving from Beta to Public Preview: list conversation messages, delete conversation message, and send thumbs up/down feedback.
  • Benchmark result explanations: Benchmark results now include an explanation for why a result was rated incorrect.
  • SQL execution result summaries: Genie now automatically generates summaries of SQL execution results to make responses more natural and easier to interpret.
  • SQL measures in knowledge stores: Users can now add SQL measures to a space's knowledge store, along with filters and dimensions, to guide Genie in generating specific SQL expressions. See Build a knowledge store for more reliable Genie spaces.
  • Fixed permissions messaging: Messaging for users with CAN RUN or CAN VIEW permissions has been corrected to accurately reflect their capabilities in the monitoring and data tabs.
  • Workspace asset tagging support: Genie spaces are now included in the workspace asset tagging public preview.
>
  • JOIN関係の発見改善: 管理が容易になるように、JOIN関係とSQL スニペット定義を並べ替えることができるようになりました。
  • APIエンドポイントがパブリックプレビューに移行: 以下のGenieAPIエンドポイントがベータから公開プレビューに移行します: 会話メッセージの一覧表示、会話メッセージの削除、および拇印アップ/ダウンフィードバックの送信。
  • ベンチマーク結果の説明: ベンチマーク結果には、結果が不正解と評価された理由の説明が含まれるようになりました。
  • SQL実行結果の概要: Genieは現在、応答をより自然で解釈しやすくするためにSQL実行結果の概要を自動的に生成します。
  • ナレッジストア内のSQLメジャー: ユーザーはスペースのナレッジストアにフィルターと次元とともにSQLメジャーを追加できるようになり、Genieが特定のSQL式を生成するのを支援できます。 信頼性の高いGenieスペースを構築するためのナレッジストアの構築をご覧ください。
  • 固定パーミッションメッセージング: CAN RUNまたはCAN VIEW権限を持つユーザーのメッセージングが、モニタリングおよびデータタブでの機能を正確に反映するように修正されました。
  • ワークスペースアセットタギングサポート: Genieスペースが、ワークスペースアセットタギングのパブリックプレビューに含まれるようになりました。

June 18, 2026

Token auto-scoping is now generally available. Auto-scoping automatically applies the principle of least privilege to personal access tokens (PATs) by observing which API endpoints a token uses and narrowing its permissions to only those APIs. Auto-scoping is on by default for new long-lived tokens (30 days or longer) and applies to existing tokens with all-APIs access based on historical usage. See Auto-scoping for personal access tokens.

> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CcCdx6QAmtwEYAzvE7XD5'}]

  • Table header color customization: Authors can now set font color, background color, and text formatting for table column headers. See Table layout settings.
  • Dashboard relationships (Public Preview): Dashboard relationships, which allow users to create complex multi-fact data models, are now in public preview. See Dashboard relationships.
  • Local metric views (GA): Local metric views, which let you define dimensions, measures, and join relationships directly in an AI/BI dashboard without publishing to Unity Catalog first, are now generally available. See Local metric views.
  • Local metric view parameters: Local metric views now support parameters, allowing Databricks to dynamically change query inputs based on user selections in a UI.
  • Local metric view wildcard expressions: Local metric views now support wildcard expressions, which allows users to inherit all fields from upstream tables or metric views.
  • Power BI enhanced report format support (Beta): Genie Code now supports Power BI files exported with the Power BI enhanced report format (PBIR).
  • Image upload suggestions for Power BI and Tableau imports (Beta): Genie Code now suggests that authors upload images when importing files from Power BI and Tableau to improve migration quality.

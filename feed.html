Databricks Docs Updates

Mount Delta shares to an existing shared catalog
> デルタマウントは既存の共有カタログに割り当てられています。
September 12, 2025

Delta Sharing recipients can now mount shares received from their Delta Sharing provider to an existing shared catalog. Previously, recipients needed to create a new catalog for each new share. See Create a catalog from a share.

> デルタ共有の受信者は、デルタ共有プロバイダーから受信したシェアを既存の共有カタログにマウントできるようになりました。以前は、受信者が新しい共有ごとに新しいカタログを作成する必要がありました。共有からカタログを作成するをご覧ください。
Serverless jobs and pipelines configured in the UI are now performance optimized by default
> サーバーレスジョブとパイプラインは、UIで構成されており、デフォルトで性能が最適化されています。
September 11, 2025

Serverless jobs and pipelines configured in the UI are now set to Performance optimized by default. This aligns the UI with existing defaults across APIs, Terraform, and SDKs. For more information, see Performance mode in serverless jobs and Performance mode in serverless pipelines.

>

サーバーレスジョブとUIで構成されたパイプラインは、デフォルトでパフォーマンス最適化に設定されています。これにより、API、Terraform、SDKにわたる既存のデフォルトがUIに合わせられます。詳細については、サーバーレスジョブのパフォーマンスモードとサーバーレスパイプラインのパフォーマンスモードをご覧ください。

AI/BI dashboard text widget and performance improvements
> AIバイBIダッシュボードのテキストウィジェットと性能の改善
September 11, 2025
  • Improved text widget editor: The dashboard text widget now has an enhanced UI for formatting text, adding links, and inserting images.
  • Fixed performance regressions: Resolved issues that caused performance slowdowns, including pages failing to load and URL parameters applying incorrectly.
>
  • テキストウィジェットエディターの改善: ダッシュボードのテキストウィジェットに、テキストの書式設定、リンクの追加、画像の挿入を行うための拡張UIが追加されました。
  • パフォーマンスの改善: ページの読み込み失敗や、URLパラメーターの適用ミスなど、パフォーマンスの低下を引き起こす問題を解決しました。
Genie chart fixes and feature enhancements
> ジーニーチャートの修正と機能強化
September 11, 2025
  • Fixed thin bars on binned date charts: Resolved an issue where very thin bars appeared in bar charts with binned dates.
  • Fixed single-point maps: Point maps now render correctly even when only a single point is present.
  • Increased value dictionary limit: Spaces can now include up to 120 value dictionaries. Newly created spaces automatically select 120 columns. For existing spaces, authors can configure settings to add more value dictionaries. See Configure value dictionaries.
  • Improved accuracy for percentage questions: Genie now returns more accurate results for percentage-type queries, such as conversion rates, with correct rounding to two digits.
  • Fixed unsaved instruction warning: The warning dialog for unsaved instruction changes now dismisses correctly.
  • Fixed cloning issue for spaces with 100 instructions: Spaces containing 100 instructions can now be cloned successfully.
  • Space ID field in settings: A new field in the settings page lets users directly copy the Genie space ID for use in building Genie space agents.
  • Upload files in Public Preview: Workspace admins can now manage the Genie - Upload File feature from the previews page. See Upload a file.
>
  • バインド日付チャートの固定細いバー: バインド日付のバーチャートで非常に細いバーが表示される問題を解決しました。
  • 単一ポイントマップの修正: ポイントマップに単一のポイントしか存在しない場合でも、正しく描画されるようになりました。
  • 値辞書の上限の増加: スペースには最大120の値辞書を含めることができるようになりました。新しく作成されたスペースでは自動的に120列が選択されます。既存のスペースでは、設定を構成して、より多くの値辞書を追加することができます。詳細は値辞書の設定を参照してください。
  • パーセンテージ質問の精度の向上: Genieは、変換率などのパーセンテージ型のクエリーに対して、2桁の正しい四捨五入で、より正確な結果を返すようになりました。
  • 保存されていない命令の警告の修正: 保存されていない命令の変更に対する警告ダイアログが正しく閉じられるようになりました。
  • 100の命令を含むスペースのクローンの問題を修正: 100の命令を含むスペースをクローンすることができるようになりました。
  • 設定ページのスペースIDフィールド: 設定ページに新しいフィールドが追加され、Genieスペースエージェントの構築に使用できるスペースIDを直接コピーできるようになりました。
  • パブリックプレビューでのファイルのアップロード: ワークスペース管理者は、プレビューページからGenie - ファイルのアップロード機能を管理できるようになりました。詳細はファイルのアップロードを参照してください。

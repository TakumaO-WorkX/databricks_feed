June 23, 2026

In July 2026, Unity AI Gateway budgets will be generally available for all accounts on AWS, Azure, and Google Cloud.

Unity AI Gateway budgets let admins define spending thresholds for requests managed through Unity AI Gateway and Genie, including shared and per-user limits, with configurable actions such as email alerting or usage blocking. See Unity AI Gateway budgets and Genie budgets.

The Unity AI Gateway budgets feature is separate from Unity AI Gateway itself, which remains in Beta. Accounts that gain access to Unity AI Gateway budgets are not automatically enrolled in Unity AI Gateway.

Databricks Docs Updates

Dashboard visualization enhancements and fixes
> ダッシュボードの視覚化の強化と修正
December 04, 2025
  • Map legend position control: Authors can now change map legend positions.
  • Individual series labels: You can now enable labels for individual series, instead of all series on the dashboard at once.
  • Fixed download format options: Resolved an issue where dashboard downloads only showed PNG format instead of multiple file formats (CSV, TSV, and Excel).
>
  • 地図凡例の位置制御: 作成者は地図凡例の位置を変更できるようになりました。
  • 個別のシリーズラベル: ダッシュボード上の全てのシリーズではなく、個別のシリーズにラベルを付けることができるようになりました。
  • 固定されたダウンロード形式オプション: ダッシュボードのダウンロードがPNG形式のみでなく、CSV、TSV、Excelなどの複数のファイル形式で表示されるようになりました。
Genie Research Agent, benchmark, and answer improvements
> ジーニー研究エージェント、ベンチマーク、回答の改善
December 04, 2025
  • Improved clarifying questions: Genie's ability to ask clarifying questions in cases of semantic ambiguity in the user's prompt has been improved.
  • Streamlined Research Agent: The Research Agent (Beta) experience has been streamlined by simplifying the thinking trace and moving hypothesis evaluations into a sidebar for easier review.
  • Benchmark error explanations: Benchmarks now include an LLM explanation of what was incorrect in the Genie generated answer.
  • Improved answer readability: When the answer summary contains all of the details in the query result, the query result table is automatically collapsed to improve readability.
  • Better instruction following: Genie now more effectively follows text instructions on how to format answer summaries to common questions.
>
  • 質問の明確化の向上: ユーザーの指示の意味的な曖昧さの場合に、Genieが質問を明確化する能力が向上しました。
  • リサーチ・エージェントの効率化: リサーチ・エージェント(ベータ版)の体験が、思考プロセスの簡素化と仮説評価をサイドバーに移動することで、より効率的になりました。
  • ベンチマークのエラー説明: ベンチマークにGenieが生成した答えの間違いを説明するLLMの説明が追加されました。
  • 答えの読みやすさの向上: 答えの要約にクエリ結果の詳細が全て含まれている場合、読みやすくするためにクエリ結果のテーブルが自動的に折りたたまれます。
  • 指示への対応力の向上: Genieは、一般的な質問への答えの要約のフォーマットに関するテキストの指示をより効果的に実行するようになりました。
Email notifications for expiring personal access tokens
> メール通知による個人アクセストークンの有効期限
December 04, 2025

Databricks will soon send email notifications to workspace users approximately seven days before their personal access tokens expire. Notifications are sent only to workspace users (not service principals) with email-based usernames. All expiring tokens within the same workspace are grouped together in a single email.

See Monitor and revoke personal access tokens.

> データブリックスは間もなく、ユーザーの個人アクセストークンの有効期限が7日前に迫った場合、ワークスペースユーザー(サービスプリンシパルではない)のメールアドレスベースのユーザー名にメール通知を送信します。同じワークスペース内の有効期限が切れるトークンはすべて単一のメールにまとめて送信されます。 個人アクセストークンの監視と取り消しを参照してください。
Dashboard visualization enhancements
> ダッシュボードビジュアライゼーションの機能強化
November 27, 2025
  • Sort by hidden measure fields: Users can now sort charts by a different measure field that does not visually appear on the bar chart. See Format axis settings.

Genie space limits and Research Agent improvements
> 人工知能アシスタントの空間制限の改善とリサーチエージェントの改善
November 27, 2025
  • Increased SQL snippets and JOIN relationship limit: The limit for SQL snippets and JOIN relationships in a Genie space has been increased to 200.
  • Research Agent starter questions: Research Agent (Beta) now displays starter questions to help users begin their data exploration.
  • Auto-suggested benchmarks: Genie now automatically suggests benchmarks based on the space context to help authors build evaluation sets.
  • Fixed knowledge store permissions: Resolved an issue where users with CAN EDIT permissions couldn't manage Genie knowledge store column descriptions and value dictionaries.
>
  • SQLスニペットとJOIN関係の制限の増加:Genieスペース内のSQLスニペットとJOIN関係の制限が200に増加しました。
  • リサーチ エージェントのスターター質問:リサーチ エージェント(ベータ)が、データ探索を開始するのに役立つスターター質問を表示するようになりました。
  • 自動提案ベンチマーク:Genieが、作成者がエバリュエーション セットを構築するのを支援するために、スペースのコンテキストに基づいて自動的にベンチマークを提案するようになりました。
  • Knowledgeストアのパーミッションが修正:CAN EDIT権限を持つユーザーでも、Genieナレッジストアのカラム記述や値辞書を管理できなかった問題が解決されました。

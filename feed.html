When you run a subset of a job's tasks, you can now expand your selection to include each task's dependencies. Add the + modifier to a task key to also run its upstream tasks (+my_task), its downstream tasks (my_task+), or both (+my_task+). Start a partial run from the UI, the REST API, the Databricks CLI, the SDKs, or Declarative Automation Bundles. See Run a subset of tasks and their dependencies.

June 30, 2026

Lakehouse Real-Time (Lakehouse//RT) is now in Beta. Lakehouse//RT is a serverless SQL warehouse type built for low-latency, high-concurrency workloads, delivering sub-second SQL read queries against Unity Catalog tables. Use it to serve analytical data to applications, run operational analytics, and power dashboards for hundreds to thousands of concurrent users. See Lakehouse Real-Time.

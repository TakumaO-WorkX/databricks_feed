Databricks Docs Updates

Dashboard visualization improvements and bug fixes
> ダッシュボードのビジュアライゼーションの改善と不具合の修正
November 20, 2025
  • Counter visualization improvements: You can now use counters to track and compare metrics over time and show sparklines. To create a counter with a sparkline, select a main value and a date column. See Counter visualization.
  • Sort by hidden measure fields: Users can now sort charts by a different measure field that does not visually appear on the bar chart. See Format axis settings.
  • Configure filter application behavior: Dashboards now support applying multiple filters at the same time alongside the default instant application. See Configure filter application behavior.
  • Fixed data label positioning: Resolved an issue where data labels were sometimes half above and half below bars on bar charts.
  • Fixed label color legibility: Resolved an issue where label color was not legible against all background colors.
>
  • カウンター可視化の改善: 時間の経過とともにメトリクスを追跡および比較し、スパークラインを表示できるようになりました。カウンターにスパークラインを作成するには、メイン値と日付列を選択します。 カウンター可視化を参照してください。
  • 非表示の測定フィールドでの並べ替え: ユーザーは、バーチャートに視覚的に表示されない別の測定フィールドでチャートを並べ替えることができるようになりました。 軸の書式設定を参照してください。
  • フィルターの適用動作の構成: ダッシュボードでは、デフォルトの即時適用に加えて、複数のフィルターを同時に適用することができるようになりました。 フィルターの適用動作を構成するを参照してください。
  • データラベルの位置の修正: バーチャートでデータラベルがときどき上部と下部に半分ずつ表示される問題を解決しました。
  • ラベルの色の判読性の修正: ラベルの色が背景色によって判読できない問題を解決しました。
Genie APIs and Research Agent in Beta
> ジーニーAPIとリサーチエージェントがベータ版で登場
November 20, 2025
  • Genie Create and Update APIs in Beta: Genie Create and Update APIs have been released to Beta. Get API has been updated to allow users to retrieve a serialized definition of the Genie space. See the Genie API.
  • Research Agent in Beta: Research Agent extends Genie's capabilities to help you uncover deeper insights and tackle complex business questions using multi-step reasoning and hypothesis testing. See Research Agent in Genie spaces.
>
  • Genieのクリエイトおよび更新APIがベータ版にリリース: Genieのクリエイトおよび更新APIがベータ版にリリースされました。Getアプリは、Genieスペースのシリアル化された定義を取得できるように更新されています。GenieのAPIをご覧ください。
  • リサーチ・エージェントがベータ版にリリース: リサーチ・エージェントは、Genieの機能を拡張し、多段階の推論とハイポテシス検証を使用して、より深い洞察を得たり複雑なビジネス上の課題に取り組むことができます。Genieスペースでのリサーチ・エージェントをご覧ください。
Draft dashboards removed from Databricks One
> ダーブリックスワンから削除されたドラフトダッシュボード
November 20, 2025

Draft dashboards no longer displayed: Only published dashboards are now visible in Databricks One. This change ensures business users see finalized content that is relevant and reliable for their workflows.

>

下書きダッシュボードは表示されなくなりました:Databricks Oneでは、公開済みのダッシュボードのみが表示されるようになりました。このChangeにより、ビジネスユーザーは、ワークフローに適切かつ信頼性の高い最終的なコンテンツを確認できるようになりました。

Configure compute size for Databricks Apps (Public Preview)
> Databricks アプリのコンピューティングサイズを構成する (パブリックプレビュー)
November 20, 2025

You can now configure the compute size for your Databricks apps to control CPU, memory, and cost based on your workload requirements. For more information, see Configure the compute size for a Databricks app.

> データブリックスアプリのCPU、メモリ、コストを、ワークロードの要件に応じて制御できるようになりました。詳細については、データブリックスアプリのコンピューティングサイズの構成をご覧ください。

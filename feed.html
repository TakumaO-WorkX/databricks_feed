Dashboard enhancements and bug fixes
> [翻訳エラー: Error code: 400 - {'type': 'error', 'error': {'type': 'invalid_request_error', 'message': 'Your credit balance is too low to access the Anthropic API. Please go to Plans & Billing to upgrade or purchase credits.'}, 'request_id': 'req_011CbWqdTNwsypmauDcVKmQd'}]
May 28, 2026
  • Banded rows in tables: Tables can now display banded rows.
  • Custom geographies in maps: Custom geographies are now available in choropleth and point maps. See Custom geographic data.
  • Genie Code for dashboard authoring: Genie Code for dashboard authoring is now generally available. See Use Genie Code for dashboard authoring.
  • Median and percentile expressions in metric views: Low-code metric view building now supports two new expression types: median and percentile. See Create and edit metric views.
  • AGGREGATE OVER enhancements: The AGGREGATE OVER operator now supports an optional INCLUSIVE or EXCLUSIVE keyword for TRAILING and LEADING ranges, and an optional OFFSET clause that shifts the range by a specified interval. See Aggregate window functions.
  • Point map shape field: Point maps now support a shape field to differentiate data points by category.
  • Default table sorting: Tables now support default sorting, allowing dashboard authors to define a default sort order for table columns.
  • Aggregated variable Y-axis fix: Fixed a bug where adding an aggregated variable to a bar, line, or area chart's Y-axis prevented adding a second field.
  • Symbol map interaction fix: Fixed symbol map interaction issues where clicking an empty area did not deselect markers and the cursor did not indicate that markers were clickable on hover.
  • Cross-filter conditional formatting fix: Fixed a bug where cross-filter hover and selection styling overrode conditional formatting colors on table cells.
  • SQL Editor results visibility fix: Fixed an issue where filtered results in the SQL Editor were not visible even though the data was present, caused by the table container collapsing within the page layout.
  • Symbol map field picker fix: Fixed an issue where the latitude and longitude field pickers in the symbol map editor did not work when using a semantic model as a dataset.
